== Installation Dependencies

Vulkan, glfw3, glm, stb, tinyobjloader.

== Build

Change `VCPKG_ROOT` in CMakeUserPresets.json.
To use vcpkg manifest mode, remove .bak extension from vcpkg*.bak files.
[source]
----
cmake --preset default
cmake --build build
----

Check build/base_render. launch the executable on the parent directory of `shader`.

Can just launch sln when using Visual Studio.
